# ☕ Dirty Cafe Sales Data Analysis 
# 📌 Overview

**This project focuses on analyzing a dirty (unclean) cafe sales dataset. The dataset contains various inconsistencies such as missing values, incorrect formats, and duplicate entries.**

*The goal is to clean, transform, and analyze the data to extract meaningful business insights.*

# Dashboard

# 📂 Dataset Description

**The dataset includes typical cafe sales information such as:**

Order ID 

Date / Time

Product Name

Category

Quantity Sold

Price

Total Sales


# ⚠️ Problems in Dataset

**Before analysis, the dataset had multiple issues:**

Missing/null values

Duplicate records

Inconsistent date formats

Incorrect data types (e.g., numbers stored as text)

Outliers in price or quantity


# 🧹 Data Cleaning Steps

**The following steps were performed:**

Removed duplicate records

Handled missing values (fill/drop depending on context)

Standardized column formats

Converted data types (e.g., string → numeric, date parsing)

Cleaned inconsistent text values

Removed or treated outliers

# 📊 Analysis Performed


**After cleaning, the following analysis was done:**

Total sales over time

Best-selling products

Category-wise revenue

Monthly sales trends

Peak sales hours

# 📈 Key Insights

**Some insights derived from the data:**

Top-performing product(s)

Highest revenue-generating category

Peak business month

Monthly growth
